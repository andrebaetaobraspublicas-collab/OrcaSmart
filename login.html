OrcaSmart para calculoobras.com.br

Sistema web para orcamentacao de obras com bases referenciais, orcamentos sinteticos, composicoes, BDI, encargos e controle por usuario.

  • Banco de dados individual para cada usuario
  • Acesso liberado por assinatura Stripe
  • Ambiente preparado para hospedagem em dominio proprio

As chaves Stripe e o preco mensal devem ser configurados no servidor. Nenhuma chave secreta fica no navegador.
